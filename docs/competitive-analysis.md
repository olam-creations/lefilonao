*Généré: 2026-02-09*

## Positionnement Marché

Le marché des alertes appels d'offres publics en France est dominé par quelques acteurs historiques.
**Le Filon AO** se positionne comme le disrupteur IA avec un pricing agressif.

---

## Concurrents Principaux

### 1. France Marchés (francemarches.com)
- **Type:** Agrégateur multi-sources (BOAMP, JOUE, 300+ journaux régionaux, 1000+ sites acheteurs)
- **Prix:** Non transparent (devis)
- **Forces:** Couverture exhaustive, marque établie
- **Faiblesses:** Prix élevé, interface datée

### 2. e-marchespublics.com
- **Type:** Plateforme de réponse + alertes
- **Prix:** Freemium + abonnements
- **Forces:** Tout-en-un (veille + réponse)
- **Faiblesses:** Complexité, orienté grands comptes

### 3. marche-public.fr
- **Type:** Portail d'information + veille
- **Prix:** Abonnements variés
- **Forces:** Contenu éducatif, communauté
- **Faiblesses:** Moins tech, UX vieillissante

### 4. Wanao / Marcoweb / Double Trade
- **Type:** Solutions entreprise
- **Prix:** 1000-5000€/an
- **Forces:** Fonctionnalités complètes, support
- **Faiblesses:** Prix prohibitif pour TPE/PME
