# FilonAO

Trouvez le filon des marches publics.

## Stack

- Next.js 15 + React 19
- Tailwind CSS 4
- Vercel deployment

## Features

- Alertes AO personnalisees
- Score Go/No-Go IA
- 3 AO/mois gratuits

## Development

```bash
npm install
npm run dev  # http://localhost:3050
```

## Deployment

```bash
vercel --prod
```
